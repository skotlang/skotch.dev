<Aside type="note">
Annotations are not yet supported by Skotch. This page is a placeholder
for when annotation parsing and code generation are implemented.
</Aside>

## Planned support

Kotlin annotations (`@Target`, `@Retention`, `@Test`, custom annotations)
will be supported in a future release. This is a prerequisite for the
`skotch test` command, which will discover test functions via `@Test`
annotations.
